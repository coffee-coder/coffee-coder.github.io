关系挖掘

标签（空格分隔）： 关系挖掘

概述

关系挖掘以商品数据，商品的关系数据、外网关系数据为基础数据源，进行不同意图的不同关系挖掘，例如品类、品牌、款式意图的同义、上下位、搭配关系。

同义关系

数据源

1. i2i同款数据源
2. 商品 + 属性数据
3. 商品详情页数据
4. 用户行为数据

算法

1. 背景: 训练样本和测试样本特征稀疏
2. 全量商品数据源 
    1. auction_info_all
    2. query_auc_entity
3. 字段
    nid ; title ; topcate ; cate
4. 提高词向量覆盖率。
5. 提高标题属性特征覆盖率。

问题

查询点击特征比较稀疏

解决方案

1. 海选 + 精选
2. 每个词都有一个候选同义词

命名实体消歧

标签（空格分隔）： WSD 算法

问题背景

随着品牌词表不断增加，在对商品进行品牌识别的时候，会出现大量的歧义。

主流统计方法

  1. 基于Bayes判别的消歧方法
    a. 基本思想：计算多义词W出现在给定上下文语境C (w1, w2, … wn)中，标注为各个义项Si概率大小P(si|C),使P(si|C)最大的义项即为该多义词W的义项标注。
    `多义词w有多个义项s1, s2, …,si…
    上下位语境C = w1, w2, …, wn
    P(si | C ) = P(C|si)P(si) / P(C )
    正确语义s = argmax P(si | C) = argmax ( P(C|si)P(si) / P(C ) )
    = argmax ( P(C|si)P(si) )
    P(C|si) = P{wj | wj }`
  1. 基于EL(entity linking)的消歧方法
  2. 基于Graph Ranking的消歧义方法
